# VoiceUp
VoiceUp - EdTech SaaS for Speech/Hearing Therapy &amp; Bilingual Support

## Goal: Help children with speech/hearing issues learn in a foreign language environment.

## Architecture: .NET 9 Microservices, Monorepo, Docker, YARP, RabbitMQ.

## Key Services: 
1. Identity (Auth), 
2. Catalog (Lessons), 
3. Study (Progress), 
4. Speech-Processing (AI Analysis), 
5. Localization (AI Translation & Bilingual support).

## Core Business Value: 
- AI speech analysis for therapy.
- Real-time translation of educational content.
- Visualizing sound for the deaf.
- Accessibility-first UI (WCAG).
